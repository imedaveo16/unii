**Najda DZ** is a modern emergency reporting application designed for the Algerian Civil Protection system. It allows citizens to report emergencies (Fire, Accidents, Medical, Rescue) with precise geolocation and media evidence.

## Features âœ¨

- **Multilingual Support**: Arabic (RTL) & English/French interfaces.
- **Geolocation**: Automatic GPS tracking with interactive Mapbox 3D maps.
- **Media Evidence**: Upload photos and record voice messages (up to 30s).
- **Offline First**: PWA support for mobile installation.
- **Real-time Clock**: Digital clock in the header.
